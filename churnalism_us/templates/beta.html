Beta Instructions

Thanks for agreeing to beta test Churnalism US!

Churnalism is a tool for finding matching fragments of text between two documents. We built up a cache of documents that includes all of Wikipedia and press releases from PR Newswire, PR News Web, EurekaAlert, congressional leadership offices, the White House, a sampling of Fortune 500 companies, prominent foundations and much more. Churnalism searches for matching fragments across all of these sources and shows you a side-by-side result.

To use the website, visit http://churnalism.sunlightfoundation.com. Enter the URL of a story or copy and paste the text into the form. That’s all there is to it! If you enter the URL and find that it didn’t pull the text you expected, this may be the result of the selected URL being behind a paywall. Try using the text copy and paste option, instead.

To use an extension, download the one for your prefered web browser by clicking its logo at http://churnalism.sunlightfoundation.com/downloads. If you use Windows 7 and have both a 32-bit version and 64-bit version of Internet Explorer installed, you’ll need an extra add-on to make the extension work in both versions. You can install that add-on here.

The extension allows you to fine tune what websites are “churned.” For instance, you can adjust your settings so that Churnalism only churns news websites and ensure that no personal information is submitted. You can edit the default whitelist of websites that will be run through Churnalism in the extension options. You can remove or edit domains from the whitelist, and there’s also a quick add box that will give you a list of domains currently open in your browser, in case you want to add them.

  • Chrome: you can access the options by clicking menu ›› Settings ›› Extensions ›› Churnalism Options.
  • Firefox: you can access the options via the Tools menu ›› Add-ons ›› Extensions ›› Churnalism Preferences ›› Options.
  • In Internet Explorer: you can simply visit the IE settings page to edit your settings.

When the Churnalism extension detects a match, a ribbon will slide down at the top of the page and prompt you to view the Churnalism or dismiss the notification. If you decide to view the Churnalism, an overlay will pop up that shows you a side-by-side view with the best match for the document. To close the pop up, click anywhere outside the overlay or hit escape.

To make sure you’ve installed the tool correctly and to be familiar with using it, here’s a few examples of positive results for matching text:

  • The universe is expanding faster than we thought
  • Can meditation make you more empathic?
  • Walk like an Egyptian: Researchers test replica big toes from 950BC - and confirm the 'world's first prosthetics' worked

In the course of testing this tool, we often received false positives when extremely long names, expanded acronyms or common phrases appeared in matches. To make the matches more meaningful, we use an algorithmic ranking to only show you matches where the density and number of matches is significant.

Thanks again for testing the beta version of Churnalism US. Please don’t hesitate to contact us with questions or comments at churnalism@sunlightfoundation.com.
